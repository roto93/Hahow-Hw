# how to start
1. `npm install -g expo-cli`

2. clone repository

`clone https://github.com/roto93/React-Native-practice.git`

3. rename the work you want
For example, if you'd like to check Hw7.

(macos):

`mv Hw7.js App.js`

4. install & start

`npm install`

`npm start -r`
